# picam: code on my PiZero home surveillance camera

## Usage

* `httpcam.py`: HTTP streaming camera
* `ndncam.py`: [HomeCam](https://yoursunny.com/p/homecam/) compatible camera

## Dependencies

    sudo apt install python3-picamera python3-pip
    sudo pip3 install autopep8 pyndn

[OpenCV 3.2.0](https://yoursunny.com/t/2018/install-OpenCV3-PiZero/)
